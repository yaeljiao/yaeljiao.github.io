代码缩进：Tab转空格的设置

发表于 2016-11-19 | 分类于 IDE

​ 公司的代码开发标准中，规定使用四个空格来进行代码缩进。多数文本编辑器都支持Tab键转空格的设置，以下收集了一些主流文本编辑器的设置方法：

Visual Studio: TOOLS – Options – Text Editor – C/C++ – Tabs, 勾选 Insert spaces 单选框，设置Tab size:4, Indent size: 4.

Notepad++: Settings – Preferences – Tab settings, 勾选Replace by space button ，设置Tab size: 4.

Source insight: Options – Document Options, 勾选Expand tabs，设置Tab width: 4.
