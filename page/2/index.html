发表于 2016-11-19 | 分类于 航空

Primus Epic 为公务机和支线客机设计，包含以下机型：

  • Embraer E-Jet family: E170 / E175 / E190 / E195
  • Embraer E-Jet E2 family: E175-E2 / E190-E2 / E195-E2
  • Dassualt Falcon F7X
  • Hawker 4000 (Horizen)
  • Gulfstream IV(GIV)
  • Agusta AW139 (AB139)

Primus Apex 基于Primus Epic ，包含以下机型：

  • Viking DHC-6(Twin Otter)
  • Haerbin Y-12F
  • Pilatus PC-12NG
  • Pilatus PC-24
  • SyberJet SJ30
  • Dornier SeaStar
  • ANTONOV An-132D(安东诺夫已解散，据称目前由沙特阿拉伯接管)
